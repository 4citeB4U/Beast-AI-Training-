# 🎓 Sovereign Architect Master Class

Welcome to the **LeeWay Standards & Agent Lee Master Class**. This curriculum is designed to transform traditional developers into elite **Sovereign System Architects**.

## 🎯 Purpose and Design

The software development landscape is rapidly shifting toward AI-native codebases. However, traditional AI agents suffer from the "Black Box" problem—they generate code without auditability, lack persistent memory, and often break their host systems because they are not governed by strict architectural rules.

**The Purpose of this Master Class** is to teach you how to build, deploy, and govern a multi-agent AI ecosystem that is **Discoverable, Auditable, Repairable, and Safe to Evolve**. You will learn how to transition from writing manual scripts to orchestrating the **7 Sovereign Agent Families** through a mandatory **8-Stage Execution Spine**.

## 🏛️ The Foundations of Sovereignty

### 1. The Governance-First Philosophy
Every Agent Lee system operates on four unbreakable pillars:
*   **Discoverable:** No hidden AI logic. Every workflow is mapped.
*   **Auditable:** The **Veritas Gate** strict-scores all outputs (85/100 minimum) before execution.
*   **Repairable:** Automatic injection of tracking headers via the `AutonomyAuditor`.
*   **Safe to Evolve:** Continuous learning via the Triple-Threat Memory System (Episodic, Semantic, Procedural).
