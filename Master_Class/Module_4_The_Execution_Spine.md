# Module 4: The Execution Spine & Sovereign Families

To prevent ecosystem chaos, the LeeWay Standard organizes the entire AI architecture into a hierarchical "Execution Spine." This structure ensures that specialized agents report correctly through their family leads up to Lee Prime.

## The Execution Spine Blueprint
The Execution Spine is a vertical structure where data transforms across the 8 stages of the Sovereign Cycle (from Perception down to Synthesis). As requests move down the spine, they are handled by specialized families.

## The 7 Governed Families
The Sovereign Ecosystem is divided into 7 core families. Specialized Agents (Atoms) belong to a family and report to a Family Lead.

1. **Nexus (Coordination & Routing)**
   *   *Role*: The managers. They route tasks to the appropriate families based on the `Structure` stage of the Sovereign Cycle.
2. **Nova (Engineering & Refactoring)**
   *   *Role*: Code execution, debugging, and integrity. Features the `NovaForge` agent for autonomous syntax fixes.
3. **Atlas (Memory & Vector Context)**
   *   *Role*: Manages the local ONNX Vector Engine, maintaining the Triple-Threat Memory (Episodic, Semantic, Procedural).
4. **Shield (Security & Compliance)**
   *   *Role*: Enforces the Veritas Gate. Contains the `AutonomyAuditor` and `StandardsAgent` to ensure the 85/100 score target is met before execution proceeds.
5. **Aura (Perception & Interface)**
   *   *Role*: Handles the UI/UX, interpreting user intent, and formatting final Markdown/React outputs.
6. **Chronos (Scheduling & Automation)**
   *   *Role*: Manages temporal tasks, background jobs, and the `Echo` stage.
7. **Prime (Final Authority)**
   *   *Role*: Features `Lee Prime`, the Final Speaker. Approves and finalizes all synthesized output.

## The Veritas Gate Pipeline
A crucial part of the Execution Spine is the **Veritas Gate**. Before any code is written to disk or any output is shown to the user, the execution thread must pass through this audit-first pipeline governed by the Shield family. If compliance is not met, the task is sent back to Nova for autonomous repair.
