# Elite Technical Flashcards (Hard Difficulty)

Use these flashcards to master the specialized terminology of the LeeWay Standards.

---

## Core Philosophy

**Card 1**
*   **Front:** What is the "Sovereignty Doctrine"?
*   **Back:** The philosophy of moving from fragmented "Black Box" AI to a unified, AI-native operating environment that is Discoverable, Auditable, Repairable, and Safe to Evolve.

**Card 2**
*   **Front:** Define "Skill Atoms".
*   **Back:** Modular, production-ready units of expertise (e.g., Code Generation, Security Auditing) that agents load dynamically. 

**Card 3**
*   **Front:** What are the three layers of the Triple-Threat Memory System?
*   **Back:** 1) Episodic (task history), 2) Semantic (persistent facts), 3) Procedural (mastery of workflows).

## The Sovereign Cycle

**Card 4**
*   **Front:** List the 8 Stages of the Sovereign Cycle in order.
*   **Back:** Perception -> Origin -> Structure -> Execution -> Veritas -> Echo -> Synthesis -> Lee Prime.

**Card 5**
*   **Front:** What is the primary function of the "Veritas" gate?
*   **Back:** A mandatory validation stage where agent outputs are audited before delivery. Outputs must meet a compliance Score Target of 85/100.

**Card 6**
*   **Front:** What does the "Echo" stage achieve?
*   **Back:** It writes the experience and outcome to episodic and semantic memory to ensure the system learns and maintains continuity.

**Card 7**
*   **Front:** Who is the "Final Speaker"?
*   **Back:** Lee Prime. The ultimate authority that delivers the unified, validated result to the user.

## The Governed Families

**Card 8**
*   **Front:** What is the role of the "Nexus" family?
*   **Back:** Coordination and Routing. They act as managers, routing tasks based on the Structure stage.

**Card 9**
*   **Front:** Which family houses the `NovaForge` agent for code refactoring?
*   **Back:** The Nova Family (Engineering & Refactoring).

**Card 10**
*   **Front:** The local ONNX Vector Engine is managed by which family?
*   **Back:** The Atlas Family (Memory & Vector Context).

**Card 11**
*   **Front:** What is the purpose of the "Authority Matrix"?
*   **Back:** Managed by Shield Aegis (Security), it enforces permissions and "Collaboration Law" between agents to prevent conflicts.

## Compliance & Architecture

**Card 12**
*   **Front:** What role does the "AutonomyAuditor" agent play?
*   **Back:** It conducts scans to prevent architectural drift and ensures the 85/100 compliance score is met before execution proceeds.

**Card 13**
*   **Front:** What is the `StandardsAgent: HeaderInjector`?
*   **Back:** An autonomous PowerShell script that injects mandatory LeeWay headers into all compliant files.

**Card 14**
*   **Front:** Why are `# REGION` and `# ENDREGION` blocks mandatory?
*   **Back:** They allow autonomous agents to surgically replace and repair code without breaking the entire file structure.

**Card 15**
*   **Front:** Name the components of the "Four-Tier Stack".
*   **Back:** 1) The Brain (FastAPI), 2) The Interface (React/Gradio), 3) The Memory (ONNX), 4) The Executors (PowerShell/Node).

**Card 16**
*   **Front:** Why do we use Docker Compose for One-Click Sovereignty?
*   **Back:** It provides isolation, one-click unified deployment, and container self-healing, ensuring the architecture is portable and repairable.
